# AI Decision Agents Portfolio

This repository contains two applied AI decision-support agents built in Python: an Ecommerce Growth Agent and a Public Health Strategy Agent. These projects demonstrate how large language models can be used to structure ambiguous problems, generate recommendations, and support business or policy decision-making.

## Projects

### Ecommerce Growth Agent
A decision-support agent designed to help ecommerce teams identify growth opportunities based on product, customer, site, and budget inputs. The agent produces recommendations across conversion, retention, merchandising, and experimentation levers.

### Public Health Strategy Agent
A decision-support agent designed to recommend intervention strategies based on population, health challenge, constraints, and available resources. The agent structures public health problems into actionable recommendations.
